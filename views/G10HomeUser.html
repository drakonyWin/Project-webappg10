Welcome to our Roomreservation system

A student can book time slot for only today. A room has four time slots: 8-10, 10-12, 13-15, 15-17 each day. If the current time is later than any time slots. Those time slots will not be available to book. If a room's time slot is booked, its status is pending. No other booking is allowed. A student can only book a single slot in one day.
